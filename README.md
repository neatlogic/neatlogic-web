neatlogic-web相关脚本

## 初始化依赖
```
npm install
```

### 本地开发环境运行项目
```
npm run serve
```

### 打包项目代码
```
npm run build
```

### 更新某个依赖
```
npm update 依赖包名字

eg:npm update techsure-ui
```


### 自动纠正文件里不符合eslint的语法（只能纠正语法不标准，错误的逻辑不会修改！！！）
```
npm run lint
```

mac的涉及到用户权限的需要在命令行前加 sudo
